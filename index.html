IoT Web Storage API

256 KiB (262144 bytes) of free disk space for your IoT devices, accessible via a simple HTTP API.

API Usage

1. Get Key

Method: GET

Endpoint: http://localhost:8080/key/

Description: Retrieves a unique key for accessing storage.

2. Read Data

Method: GET

Endpoint: http://localhost:8080/api/get/{key}={offset}+{length}

Example: http://localhost:8080/api/get/LZc75h7xoLIcrL8zDcdQRMTkioHPdu56WiZKXvRpZCc=0+32

Key: LZc75h7xoLIcrL8zDcdQRMTkioHPdu56WiZKXvRpZCc=

Offset: 0

Length: 32

Description: Reads 32 bytes from the start of the block.

3. Write Data

Method: POST

Endpoint: http://localhost:8080/api/get/{key}={offset}+{length}

Example: http://localhost:8080/api/get/LZc75h7xoLIcrL8zDcdQRMTkioHPdu56WiZKXvRpZCc=5+538

Key: LZc75h7xoLIcrL8zDcdQRMTkioHPdu56WiZKXvRpZCc=

Offset: 5

Length: 538

Description: Writes data starting at offset 5 for 538 bytes.

Caution: The length of the POST data should match the specified length in the URL.
